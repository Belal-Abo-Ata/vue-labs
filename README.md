## Installation

To run this project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/Belal-Abo-Ata/vue-labs

    Navigate to the project directory:

    bash

cd vue-labs

Install dependencies:

bash

npm install

Run the development server:

bash

    npm run serve

    Open your browser and visit http://localhost:8080 to view the app.

Usage

    Navigate to the "Users" page to view a list of users.
    Click on "Add User" to create a new user.
    Click on a user to view their details and make changes.
    To delete a user, click on the delete icon next to their name.

Technologies Used

    Vue.js
    Vue Router
    Vuex (for state management)
    Bootstrap (for styling)

API Integration

This project integrates with a backend API for managing users. Make sure to update the API endpoints in the src/services/userService.js file to match your backend API.
